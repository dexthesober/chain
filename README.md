# React Account Manager 

This is a small React (Vite) application where user can do  registration, login, and profile edit functionality. It is frontend-only: user data is stored in the browser's localStorage for demo purposes.

## Run locally
1. Install dependencies:

npm install

2. Start dev server:

npm run dev

3. Open the URL shown (default http://localhost:5173)

## Notes

- The project uses Bootstrap 5 (CDN) for layout and basic styling.
